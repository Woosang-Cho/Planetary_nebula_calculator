# planetary_nebula_calculator

Radial velocity of PN calculator made of Python3.

## 0. Radial velocity(시선속도)
**시선속도** 란 율리우스력의 기원전 4713년 1월 1일 월요일 정오(UTC 기준)를 기점으로 계산한 날짜 수입니다.
